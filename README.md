## 📝 구현 목록

- 과제 A: To-Do List App 만들기 (UI)
    [o][필수] 적절한 Header를 만든다.
    [o][필수] 투두리스트에 적합한 기능을 구현하기 위해 **데이터를 조작**할 수 있다.
    [o][필수] 스크롤시 Header가 사라지지 않고 화면 상단에 고정되도록 한다.
    [o][필수] 필수적으로 추가해야할 기능: Task 목록 조회, 새로운 Task 추가, Task 삭제
    [o][필수] 투두리스트에 적절한 애니메이션을 추가할 수 있다.
        [필수] Drag and Drop으로 Task의 순서를 변경한다.
               데이터를 변경하지 않고 화면 내에서 Task의 순서만 변경되면 됨
    [x][선택] 필수 구현 항목에 덧붙여 필요한 **데이터 속성을 추가하여 정의**할 수 있다
    [x][선택] 최소 요구사항에 덧붙여 구현하고 싶은 기능이 있으면 추가적으로 구현.
    [x][선택] 최소 요구사항에 덧붙여 추가하고 싶은 UI/UX 및 애니매이션을 추가적으로 구현.

- 과제 B: To-Do List App만들기 (Data)
    [o][필수] 투두리스트에 적합한 기능을 구현하기 위해 데이터를 조작할 수 있다.
    [o][필수] 필수 기능:
        - Task 목록 조회
        - 새로운 Task 추가
        - Task 삭제
    [o][필수] 최소 두가지 이상의 조건으로 Task를 필터링 (ex. 상태, 생성일, 생성자, 중요도)
    [o][필수] Task의 상태 변경 (ex. 진행중 → 완료)
    [o][선택] 최소 요구사항에 덧붙여 필요한 데이터 속성을 추가하여 정의할 수 있다
    [o][선택] 최소 요구사항에 덧붙여 구현하고 싶은 기능이 있으면 추가적으로 구현.
    [o][선택] 최소 요구사항에 덧붙여 추가하고 싶은 투두리스트에 적절한 UI/UX를 추가할 수 있다.
